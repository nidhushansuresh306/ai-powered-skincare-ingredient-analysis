# AI-Powered Skincare Ingredient Safety Analysis

## Overview

AI-Powered Skincare Ingredient Safety Analysis is a smart web-based system designed to analyze the chemical ingredients present in skincare and cosmetic products. The system helps users identify potentially harmful, toxic, or skin-irritating ingredients used in creams and other skincare products.

The project uses Artificial Intelligence techniques to evaluate ingredient safety and provide users with insights about possible side effects and skin impact.

---

## Features

* Analyze skincare and cosmetic ingredients
* Detect harmful or unsafe chemical compounds
* Provide ingredient safety information
* User-friendly interface
* Fast ingredient analysis system
* Database management using MySQL

---

## Technologies Used

* PHP
* HTML
* CSS
* JavaScript
* MySQL
* Artificial Intelligence Concepts

---

## Project Objective

The main objective of this project is to help users make safer skincare choices by understanding the ingredients used in cosmetic creams and skincare products.
